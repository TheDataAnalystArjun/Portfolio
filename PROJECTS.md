# Detailed Projects Overview

## Machine Learning Projects

### 1. Car Insurance Fraud Detection
- **Objective**: Detect fraudulent insurance claims using ML algorithms
- **Dataset**: Insurance claim data with labeled fraud cases
- **Models Used**: Logistic Regression, Random Forest, XGBoost
- **Results**: 95% accuracy achieved
- **Key Features**: Claim amount, claimant history, claim type

### 2. Movie Reviews Sentiment Classification
- **Objective**: Classify movie reviews as positive or negative
- **Dataset**: IMDB movie reviews
- **Techniques**: NLP, Text preprocessing, Feature extraction
- **Models Used**: Naive Bayes, SVM, Neural Networks
- **Results**: High accuracy in sentiment classification

### 3. Spam Mail Classification
- **Objective**: Identify spam emails with high precision
- **Dataset**: Email dataset with spam labels
- **Features**: Email content analysis, sender information
- **Models Used**: Logistic Regression, Decision Trees
- **Results**: Efficient spam detection system
